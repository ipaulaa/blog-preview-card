# Blog Preview Card

This is a solution to the [Blog Preview Card](https://www.frontendmentor.io/challenges/blog-preview-card-ckPaj01IcS)
challenge on [Frontend Mentor](https://www.frontendmentor.io)

## 💪 Challenge

Create a Blog Preview Card and style it according to the provided design specifications.
The component should be visually appealing and fully responsive, ensuring optimal functionality
across different devices and screen sizes. Additionally, you need to implement **hover** and
**focus** states for all interactive elements on the card.
